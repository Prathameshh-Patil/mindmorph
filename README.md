# 🍥 MindMorph

**MindMorph** is an AI-powered full-stack application that transforms your simple hand-drawn doodles into stunning, unique digital creatures. Using Gemini Vision AI for analysis and FLUX AI for high-fidelity image generation, MindMorph brings your imagination to life in seconds.

🚀 **Live App:** [mindmorphh.vercel.app](https://mindmorphh.vercel.app/)  
📡 **Backend API:** [mindmorph-y4vz.onrender.com](https://mindmorph-y4vz.onrender.com/)

---

## ✨ Features

- **Interactive Canvas**: Draw your ideas directly in the browser.
- **AI Vision Analysis**: Powered by Google Gemini 1.5 Flash to interpret the mood, energy, and traits of your doodle.
- **Generative AI Art**: Uses the FLUX model to paint a high-definition creature based on your drawing.
- **Global Gallery**: Explore and "Like" creatures generated by other users around the world.
- **Persistence**: All creatures and their traits are stored in a Neon PostgreSQL database.

---

## 🛠️ Tech Stack

- **Frontend**: React (Vite), Tailwind CSS, Lucide React
- **Backend**: Node.js, Express.js (ES Modules)
- **Database**: Neon (PostgreSQL)
- **AI Models**: 
  - **Google Gemini 1.5 Flash** (Image-to-JSON analysis)
  - **FLUX AI** via Pollinations (Prompt-to-Image generation)
- **Deployment**: Vercel (Frontend), Render (Backend)

## 🚀 Local Setup

### 1. Prerequisites
- Node.js (v18+)
- A Neon PostgreSQL account
- A Google Gemini API Key

### 2. Backend Configuration
Navigate to `/server` and create a `.env` file:
```env
PORT=3001
DATABASE_URL=your_neon_postgres_url
GEMINI_API_KEY=your_google_ai_key
```

### 3. Frontend Configuration
Navigate to `/client` and create a `.env` file:
```env
VITE_BACKEND_URL=http://localhost:3001
```

### 4. Installation & Running
```bash
# In one terminal (Backend)
cd server
npm install
npm run dev

# In another terminal (Frontend)
cd client
npm install
npm run dev
```
